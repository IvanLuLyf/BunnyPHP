# BunnyPHP

BunnyPHP is a simple MVC Framework.

## Directory Structure

* ```index.php``` Entry File

* ```app``` : Your Application Dir
* * ```controller``` : Controller Dir
* * ```model``` : Model Dir
* * ```service``` : Service Dir
* * ```BunnyPHP``` Framework Dir
* ```config``` Configure Dir
* ```static``` Static Files' Dir
* ```template``` Template Files' Dir
